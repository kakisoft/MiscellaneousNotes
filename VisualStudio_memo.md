# Visual Studio メモ

## クイックウォッチ呼び出し
Ctrl + d → q

## 呼び出し元のメソッド一覧を表示
```
shift + F12

極端に抽象化が進み、トレース不可となっていた場合、この方法で。
```


#### ショートカット

|　　選択                               |　　キー                    |
|:--------------------------------------|:---------------------------|
|  大文字に変換                         |  Ctrl + Shift + u          |
|  小文字に変換                         |  Ctrl + u                  |
|  フォルダを指定して検索               |  Ctrl + Shift + f          |
|  クイックウォッチ                     |  Ctrl d → q               |
|  リネーム                             |  選択→ Ctrl + r → r      |
|  メソッド切り出し                     |  選択→ Ctrl +  r → m     |



## 設定
ツール→オプション→環境→キーボード→「Visual C# 2005」

ツール→コードスニペットマネージャー

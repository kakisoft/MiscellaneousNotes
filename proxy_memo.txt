【git】

＜設定＞
git config --global http.proxy http://[ID:PASSWORD]@[プロキシサーバのURL]:[ポート番号]

git config --list

＜設定解除＞
git config --global --unset http.proxy
git config --global --unset https.proxy
git config --global --unset url."https://".insteadOf git://
